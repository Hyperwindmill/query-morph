# query-morph

A modern TypeScript library for structural query transformations.

## Getting Started

### Installation

```bash
npm install query-morph
```

### Usage

```typescript
import { greet } from 'query-morph';

console.log(greet({ name: 'World' }));
```

## Development

### Build

```bash
npm run build
```

### Test

```bash
npm run test
```

### Lint

```bash
npm run lint
```
