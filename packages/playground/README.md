# @morphql/playground

An interactive, real-time web editor to test and visualize **Morph Query Language** (MorphQL) transformations.

## Features

- **Live Preview**: See transformation results instantly as you type.
- **Code Generation**: Visualize the pure JavaScript function generated by the engine.
- **Monaco Editor**: Rich editing experience with syntax highlighting.
- **Format Toggle**: Test conversions between JSON and XML on the fly.

## Development

To start the playground locally during development:

```bash
npm run playground
```

Wait for changes to propagate from `@morphql/core` automatically.

## Structure

The playground is built with:

- **React 19**
- **Vite**
- **Tailwind CSS 4**
- **Monaco Editor**
